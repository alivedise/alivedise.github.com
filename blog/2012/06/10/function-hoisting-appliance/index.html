by Alive Kuo - Jun 10th, 2012 - posted in javascript

function hoisting是什麼？

一個不說你也不會知道的js語言特性，是指js看到function xxx(){}這種宣告時會把定義搬到scope最頂端 的行為，hoisting的意思是提昇。
